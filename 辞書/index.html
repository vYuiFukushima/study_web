辞書とは

辞書（Dictionary）は、 キー（Key）と値（Value）をセットで管理するデータ型です。 

dct = {
    "id": "0001",
    "name": "guest"
}

辞書の作り方 {キー1:値1, キー2:値2, ...} 

fruits = {
    1: "Orange",
    2: "Lemon",
    3: "Peach"
}
colors = {
    "red": "赤色",
    "blue": "青色",
    "yellow": "黄色"
}
user = {
    "old": 28,
    "address": "Tokyo"
}

キーを指定して値を取得する 辞書[キー] 

mydict = {
    "JP": "Japan",
    "DE": "Germany",
    "FR": "France"
}

print(mydict["DE"])
実行結果：Germany

getメソッドで値を取得する 辞書.get(key[, default]) 

mydict = {
    "JP": "Japan",
    "DE": "Germany",
    "FR": "France"
}

print(mydict.get("DE"))
print(mydict.get("EN"))
print(mydict.get("EN", "NotFound"))
実行結果
Germany
None
NotFound

辞書の要素の値を変更する辞書[キー] = 値

既に存在するキーに対して値を代入すると、 そのキーの値を変更できます。 

mydict = {
    1: "Movie",
    2: "Foods",
    3: "Reading"
}

mydict[2] = "Sports"

print(mydict)
実行結果
{1: 'Movie', 2: 'Sports', 3: 'Reading'}

辞書に新しい要素を追加する 辞書[キー] = 値

存在しないキーに対して値を代入すると、 新しい要素が追加されます。 

mydict = {
    1: "Movie",
    2: "Foods",
    3: "Reading"
}

mydict[4] = "Sports"

print(mydict)
実行結果
{1: 'Movie', 2: 'Foods', 3: 'Reading', 4: 'Sports'}

辞書に含まれるすべてのキーを取得する 辞書.keys()

keys() を使うと、 辞書に含まれるすべてのキーを取得できます。 

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

print(mydict.keys())
実行結果
dict_keys(['L', 'O', 'G'])

for文でキーを順番に取り出す

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

for mykey in mydict.keys():
    print(mykey)
実行結果
L
O
G

辞書に含まれるすべての値を取得する 辞書.values()

values() を使うと、 辞書に含まれるすべての値を取得できます。 

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

print(mydict.values())
実行結果
dict_values(['Lemon', 'Orange', 'Grapes'])

for文で値を順番に取り出す

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

for myvalue in mydict.values():
    print(myvalue)
実行結果
Lemon
Orange
Grapes

辞書に含まれるすべてのキーと値を取得する 辞書.items()

items() を使うと、 キーと値の組み合わせを取得できます。 

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

print(mydict.items())
実行結果
dict_items([('L', 'Lemon'), ('O', 'Orange'), ('G', 'Grapes')])

for文でキーと値を順番に取り出す

mydict = {
    "L": "Lemon",
    "O": "Orange",
    "G": "Grapes"
}

for mykey, myvalue in mydict.items():
    print("key:" + mykey + ", value:" + myvalue)
実行結果
key:L, value:Lemon
key:O, value:Orange
key:G, value:Grapes

del文を使って要素を削除する del 辞書[キー]

del を使うと、 指定したキーの要素を削除できます。 

mydict = {
    "A": "Apple",
    "L": "Lemon",
    "O": "Orange"
}

del mydict["L"]

print(mydict)
実行結果
{'A': 'Apple', 'O': 'Orange'}

指定したキーの要素を削除する 辞書.pop(key[, default])

pop() は要素を削除すると同時に、 削除した値を取得できます。 

mydict = {
    "A": "Apple",
    "L": "Lemon",
    "O": "Orange"
}

val = mydict.pop("L")

print(val)
print(mydict)
実行結果
Lemon

{'A': 'Apple', 'O': 'Orange'}

最後に追加された要素を取得して削除する 辞書.popitem()

popitem() は、 最後に追加されたキーと値を取得しながら削除します。 

mydict = {
    "A": "Apple",
    "L": "Lemon",
    "O": "Orange"
}

val = mydict.popitem()

print(val)
print(mydict)
実行結果
('O', 'Orange')

{'A': 'Apple', 'L': 'Lemon'}

続けて実行した場合

val = mydict.popitem()
print(val)

val = mydict.popitem()
print(val)

print(mydict)
実行結果
('L', 'Lemon')
('A', 'Apple')

{}

すべての要素を削除する 辞書.clear()

clear() を使うと、 辞書のすべての要素を削除できます。 

mydict = {
    "A": "Apple",
    "L": "Lemon",
    "O": "Orange"
}

mydict.clear()

print(mydict)
実行結果
{}

CircleCI Home

CircleCI Home画面

Organization Homeとは

組織（Organization）全体の状況を確認するトップページ。

CircleCIに登録されているプロジェクトの実行状況や利用状況を一覧で確認できる。

プロジェクト一覧

Home画面中央には、登録されているプロジェクトが表示される。

例：

circleci-practice

各プロジェクトから以下の画面へ移動できる。

  • Overview
  • Pipelines

circleci-practice

7 minutes ago
LAST TRIGGERED BY

circleci-practice2

17 hours ago

circleci-project-setup

Overview

プロジェクトの概要を確認する画面。

以下のような情報を確認できる。

  • プロジェクトの実行状況
  • 最近の成功・失敗状況
  • 利用状況

Pipelines

プロジェクトの実行履歴を確認する画面。

GitHubへのPushなどをきっかけに実行されたパイプラインの履歴が表示される。

例：

1回目の実行 → 成功
2回目の実行 → 失敗
3回目の実行 → 成功

Last Run

最後にWorkflow（パイプライン）が実行された日時。

例：

7 minutes ago

↓

7分前に最後の実行が行われた

ことを表す。

Last Triggered By

最後にWorkflowを実行したブランチ。

例：

main

↓

mainブランチの変更によって実行された

ことを表す。

ブランチとは

ソースコードの作業場所を分ける仕組み。

例：

main
develop
feature/login

実行結果

成功

緑色のチェックマーク

✓

Workflowが正常終了したことを表す。

失敗

赤色のマーク

✕

Workflowが失敗したことを表す。

用語

Workflow

CircleCIで実行される一連の処理。

例：

GitHubへPush
↓
CircleCI起動
↓
テスト実行
↓
結果表示

Billing Period

利用状況を集計する請求期間。

クレジット使用量やアクティブユーザー数の集計単位として利用される。

Insights

CircleCIの利用状況や統計情報を確認する画面。

Throughput

過去30日間のWorkflow実行回数の1日平均。

例：

24

↓

過去30日間で1日平均24回Workflowが実行された

ことを表す。

See all insights

利用状況や統計情報の詳細画面を表示する。

People

CircleCIを利用しているユーザーやチームメンバーを管理する画面。

Active users this billing period

現在の請求期間中にCircleCIを利用したユーザー数。

例：

0

↓

現在の請求期間中に利用したユーザーは0人

を表す。

Add more teammates

CircleCIへ新しいメンバーを招待する。

Plan

契約プランと利用状況を確認する画面。

Usage this billing period

現在の請求期間における利用状況。

例：

Free

↓

無料プランを利用中

を表す。

Credit Usage

CircleCIで消費したクレジット数を表示する。

例：

420 used
29,580 remaining

↓

420クレジット使用済み
29,580クレジット残り

を表す。

Creditとは

CircleCIでジョブを実行する際に消費される利用ポイント。

実行時間や利用リソースに応じて消費される。

See more usage data

利用状況の詳細画面を表示する。

確認できる内容：

  • クレジット使用履歴
  • ジョブ実行履歴
  • 利用状況の推移
  • プラン利用状況

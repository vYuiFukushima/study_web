CircleCI Projects

Projects画面

Projectsとは

CircleCIで利用可能なプロジェクトを管理する画面。

GitHubなどのバージョン管理システムと連携しているプロジェクト（リポジトリ）の一覧を確認できる。

ここから以下の操作を行う。

  • CircleCIのセットアップ
  • プロジェクトのフォロー
  • プロジェクト概要の確認
  • Pipeline履歴の確認

利用可能なプロジェクトが一覧表示される。

例：

Project name

プロジェクト名で検索する機能。

例：

study_web
dashboard
circleci-practice2

Overview

プロジェクトの概要画面へ移動する。

確認できる内容：

  • プロジェクト概要
  • 実行状況
  • 最近の実行結果

Pipelines

プロジェクトのPipeline実行履歴画面へ移動する。

確認できる内容：

  • 実行履歴
  • 実行結果
  • 実行時間
  • 実行ブランチ

Projects画面の役割

Projects画面は、CircleCIで利用するプロジェクトを管理するための画面である。

主な用途：

  • 新しいプロジェクトをCircleCIへ登録する
  • 既存プロジェクトをフォローする
  • プロジェクト概要を確認する
  • Pipeline実行履歴を確認する

Set up

CircleCIをそのプロジェクトで利用できるように設定する。

実行するとCircleCIのセットアップ画面が表示される。

Select your config.yml file

利用するCircleCI設定ファイル（config.yml）を選択する。

config.ymlとは

CircleCIの実行内容を定義する設定ファイル。

例：

jobs:
  test:
    steps:
      - run: pytest

config.ymlの選択方法

Fastest

既にリポジトリ内に存在する

.circleci/config.yml

を利用する方法。

Select a branch with config.yml

config.ymlが存在するブランチを選択する。

例：

main
develop
feature/test

Faster

CircleCIがサンプルの設定ファイルを作成し、新しいブランチへコミットする方法。

Fast

編集可能なconfig.ymlテンプレートを利用する方法。

CircleCIが用意したテンプレートを編集して利用する。

Set Up Project

選択した内容でCircleCIの設定を開始する。

Follow / Unfollow

Follow

プロジェクトをフォローする。

フォローするとHome画面に表示される。

Unfollow

フォローを解除する。

Home画面から非表示になる。

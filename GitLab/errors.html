エラー対処

ファイルが見つからないときや、コミットできないときの確認ポイントです。

[Errno 2] No such file or directory

意味 指定されたファイル、またはフォルダが見つかりません。

確認

git status

管理していない状態の表示例

Untracked files:

03_slack_messagenger.py
report 1.json

Gitは「そのファイルがあるのは知っているけど、まだ管理していないよ」という状態です。ファイルの追加は「03. 実行方法」のファイル追加を参照します。

git commitできない

発生例

Changes not staged for commit:

modified: 04_veriGen_reporter.py

原因

ファイルを変更したが、git addを実行していない状態です。

対応

git add .

git commit -m "任意のメッセージ"

git push

特定のファイルだけ追加したい場合は、git add ファイル名を使います。

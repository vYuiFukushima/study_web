プロジェクトを新規作成

GitLabでプロジェクトを作成し、ローカルのC:\workへクローンしてVS Codeで開くまでの流れです。

GitLab側でプロジェクトを作る

  1. プロジェクトの名前を入力する。
  2. プロジェクトのURLで、グループまたはネームスペースを選択する。
  3. プロジェクトの可視性は、表示レベルを非公開にする。
  4. プロジェクトの初期設定で「リポジトリを初期化しREADMEファイルを生成する」にチェックを入れる。
  5. プロジェクトを作成ボタンをクリックする。

ローカルへクローン

PowerShellで作業フォルダへ移動してから、GitLabのURLを使ってクローンします。

URLからコマンドを作成

cd C:\work

git clone <GitLabのURL>

URLはGitLabでプロジェクトを開き、「コード」という青いボタンを押すと表示されます。

VS Codeで開く

  1. C:\workにフォルダが作成されていることを確認する。
  2. そのフォルダをVS Codeで開く。

scriptの書き方

scriptには、PowerShell、Linuxシェル、コマンドプロンプトで手動実行するコマンドを1行ずつ書きます。

ポイント

GitLab Runnerが実行するコマンドを、script: の下にリストとして並べます。

script:
  - echo "開始"
  - python test.py

基本コマンド

echo

文字を表示します。

script:
  - echo "開始"

python

Pythonスクリプトを実行します。

script:
  - python test.py

pytest

テストを実行します。

script:
  - pytest test.py

pip

ライブラリをインストールします。

script:
  - pip install requests

または、Python経由でpipを呼び出します。

script:
  - python -m pip install requests

ファイル確認系

詳しくはこちら

dir（Windows）

ファイル一覧を表示します。

script:
  - dir

ls（Linux）

Linux環境でファイル一覧を表示します。

script:
  - ls

pwd

現在のフォルダを表示します。

script:
  - pwd

結果例

/builds/project

Git系

詳しくはこちら

git status

script:
  - git status

git log

script:
  - git log --oneline

Playwright系

ブラウザインストール

script:
  - playwright install chromium

テスト実行

script:
  - pytest hotel_test.py

便利なデバッグ用

Python確認

script:
  - python --version

結果例

Python 3.11.6

pip確認

script:
  - pip --version

環境変数確認（Linux）

script:
  - env

環境変数確認（Windows）

script:
  - Get-ChildItem Env:

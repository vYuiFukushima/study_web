関数とは

関数とは、処理をひとまとめにして 名前を付けたものです。

同じ処理を何度も実行したい場合に利用します。

関数を定義する def 関数名(): 

def hello():
    print("こんにちは")

def を使って関数を定義します。

関数を呼び出す 関数名() 

def hello():
    print("こんにちは")

hello()
実行結果
こんにちは

引数を使う def 関数名(引数): 

def hello(name):
    print("こんにちは", name)

hello("太郎")
実行結果
こんにちは 太郎

戻り値を返す return 

def add(x, y):
    return x + y

result = add(10, 20)

print(result)
実行結果
30

複数の引数を使う def 関数名(引数1, 引数2)

引数はカンマ区切りで複数指定できます。 

def add(x, y):
    print(x + y)

add(10, 20)
実行結果
30

デフォルト引数 引数 = 初期値

引数に値が渡されなかった場合に使用する値を設定できます。 

def hello(name="ゲスト"):
    print("こんにちは", name)

hello()
hello("太郎")
実行結果
こんにちは ゲスト
こんにちは 太郎

キーワード引数 引数名 = 値

引数名を指定して値を渡せます。 

def profile(name, age):
    print(name, age)

profile(age=20, name="太郎")
実行結果
太郎 20

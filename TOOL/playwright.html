About

Playwrightとは

Playwright（プレイライト）は、Webブラウザを自動で操作できるツールです。

人がブラウザで行う操作（クリック、文字入力、画面遷移など）をプログラムで自動実行できます。

主な用途

  • Webアプリケーションのテスト
  • 繰り返し作業の自動化
  • 動作確認の効率化

Scope

できること・できないこと

できること

ブラウザ操作を自動化する

ブラウザの基本操作

  • クリック
  • 文字入力
  • チェックボックス選択
  • プルダウン選択
  • ページ遷移

画面内容の取得

  • 文字列の取得
  • ページタイトルの取得
  • URLの取得

その他

  • 自動的な待機処理
  • スクリーンショットの取得
  • 動画の取得
  • 実際のブラウザ操作をスクリプトに変換（Codegen）
  • Chrome、Edge、Firefox、Safariでの実行

「画面に○○が表示されるまで待つ」「ボタンが押せる状態になるまで待つ」といった待機処理も扱えます。

できないこと

ブラウザ以外は基本的に対象外

テストスクリプトの自動修復

画面変更によってテストが失敗した場合、セレクタの変更をPlaywrightが自動で直すことは基本的にできません。

ボタン名が変わった セレクタが変わった 自動で修正

ブラウザ外の操作

Playwrightは主にブラウザ操作用のツールです。以下のような操作は対象外です。

  • Excel操作
  • メモ帳操作
  • Windowsアプリ操作
  • ファイルエクスプローラー操作

Syntax

Playwrightの基本文法

よく使う操作を、文法と例で確認します。

Webページを開く

page.goto("URL")

例

page.goto("https://example.com")

文字入力

page.fill("セレクタ", "入力したい文字")

例

page.fill("#username", "テスト太郎")

クリック

page.click("セレクタ")

例

page.click("#submit-button")

チェックボックスを選択

page.check("セレクタ")

例

page.check("#sightseeing")

プルダウンを選択

page.select_option("セレクタ", "選択する値")

例

page.select_option("#contact", "no")

キーボード操作

page.press("セレクタ", "キー")

例

page.press("#date", "Tab")

スクリーンショットを保存

page.screenshot(path="保存先")

例

page.screenshot(path="sample.png")

現在のURLを取得

page.url

例

current_url = page.url

ページタイトルを取得

page.title()

例

title = page.title()

テキストを取得

page.text_content("セレクタ")

例

message = page.text_content("#message")

Flow

よく使う流れ

# ページを開く
page.goto("URL")

# 入力する
page.fill("セレクタ", "値")

# ボタンを押す
page.click("セレクタ")

# 結果を確認する
page.text_content("セレクタ")

# スクリーンショットを取得する
page.screenshot(path="sample.png")

基本の考え方

Playwrightのテストは基本的に、次の流れで記述します。

  1. 開く
  2. 入力する
  3. 操作する
  4. 確認する

About

Playwrightとは

Playwright（プレイライト）は、Webブラウザを自動で操作できるツールです。

人がブラウザで行う操作（クリック、文字入力、画面遷移など）をプログラムで自動実行できます。

主な用途

  • Webアプリケーションのテスト
  • 繰り返し作業の自動化
  • 動作確認の効率化

Scope

できること・できないこと

できること

ブラウザ操作を自動化する

ブラウザの基本操作

  • クリック
  • 文字入力
  • チェックボックス選択
  • プルダウン選択
  • ページ遷移

画面内容の取得

  • 文字列の取得
  • ページタイトルの取得
  • URLの取得

その他

  • 自動的な待機処理
  • スクリーンショットの取得
  • 動画の取得
  • 実際のブラウザ操作をスクリプトに変換（Codegen）
  • Chrome、Edge、Firefox、Safariでの実行

「画面に○○が表示されるまで待つ」「ボタンが押せる状態になるまで待つ」といった待機処理も扱えます。

できないこと

ブラウザ以外は基本的に対象外

テストスクリプトの自動修復

画面変更によってテストが失敗した場合、セレクタの変更をPlaywrightが自動で直すことは基本的にできません。

ボタン名が変わった セレクタが変わった 自動で修正

ブラウザ外の操作

Playwrightは主にブラウザ操作用のツールです。以下のような操作は対象外です。

  • Excel操作
  • メモ帳操作
  • Windowsアプリ操作
  • ファイルエクスプローラー操作

Syntax

Playwrightの基本文法

pytestでPlaywrightを書くときの基本形です。

テストコードのひな型

from playwright.sync_api import Page
import pytest

class テスト名(object):

    @pytest.fixture(scope="function", autouse=True)
    def page_fixture(self, page: Page):
        self.page = page
        yield
        self.page.close()

    def test_change_all_params(self):
        page = self.page
        page.goto("URLをここに書く", wait_until="networkidle")

        # 操作内容をここに書いていく

        # キーボード操作
        page.press("セレクタ", "キー")

        # 文字入力
        page.fill("セレクタ", "入力したい文字")

        # クリック
        page.click("セレクタ")

        # チェックボックスを選択
        page.check("セレクタ")

        # プルダウンを選択
        page.select_option("セレクタ", "値")

        # スクリーンショットを保存
        page.screenshot(path="保存先")

        # Webページを開く
        page.goto("URL")

        # 現在のURLを取得
        page.url

        # ページタイトルを取得
        page.title()

        # テキストを取得
        page.text_content("セレクタ")

Flow

ホテルサイトのサンプルコード

from playwright.sync_api import Page
import pytest

class TestHotelPlanisphere(object):

    # 前後の処理
    @pytest.fixture(scope="function", autouse=True)
    def page_fixture(self, page: Page):
        self.page = page
        yield
        self.page.close()

    def test_change_all_params(self):
        page = self.page
        page.goto("https://hotel-example-site.takeyaqa.dev/ja/reserve.html?plan-id=0", wait_until="networkidle")

        # 宿泊日を入力
        page.fill("#date", "2021/12/06")
        page.press("#date", "Tab")

        # 宿泊日数を入力
        page.fill("#term", "3")

        # 人数を入力
        page.fill("#head-count", "2")

        # お得なプランを選択
        page.check("#sightseeing")

        # お名前を入力
        page.fill("#username", "テスト太郎")

        # 確認のご連絡を選択
        page.select_option("#contact", "no")

        # 予約内容を確認するボタンをクリック
        page.click("#submit-button")

        # スクリーンショットの保存
        page.screenshot(path="スクリーンショットのパス")

基本の考え方

Playwrightのテストは基本的に、次の流れで記述します。

  1. 開く
  2. 入力する
  3. 操作する
  4. 確認する

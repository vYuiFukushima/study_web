Webブラウザ

Playwright

何をするもの？

Webブラウザを自動操作するツールです。人がブラウザで行う操作を自動化できます。

できること

  • ログイン
  • 検索
  • ボタン押下
  • 画面遷移確認
  • 入力チェック

操作対象

  • Chrome
  • Edge
  • Firefox
  • Safari

特徴

  • Microsoft製
  • 比較的新しい
  • 動作が速い
  • 自動待機機能が優秀

向いているもの：Webシステム、Webアプリ、ECサイト

Webブラウザ

Selenium

何をするもの？

Webブラウザを自動操作するツールです。長年利用されている定番のWebテストツールです。

できること

  • ログイン
  • 検索
  • ボタン押下
  • データ入力
  • 画面確認

操作対象

  • Chrome
  • Edge
  • Firefox

特徴

  • 歴史が長い
  • 利用企業が多い
  • 情報量が豊富

向いているもの：Webシステム、Webアプリ

昔から使われている定番Webテストツール

Playwright と Selenium の違い

項目 Playwright Selenium
新しさ 新しい 古い
速度 速い やや遅い
自動待機 強い 弱い
情報量 多い 非常に多い
利用実績 増加中 非常に多い

API Test

APIテスト

API

Postman

何をするもの？

APIを実行して動作確認を行うツールです。画面を操作するのではなく、APIへ直接リクエストを送ります。

できること

  • GET実行
  • POST実行
  • PUT実行
  • DELETE実行
  • レスポンス確認
  • ステータスコード確認

操作対象

API

GET /users
POST /login
PUT /books/1
DELETE /books/1

特徴

  • API開発の定番ツール
  • 開発者やテスターがよく利用する
  • 自動実行（Newman）にも対応

向いているもの：APIテスト、API開発
